== Description ==

This plugin passes all payments to Factuursturen and creates automatic invoices. Works with WooCommerce payments.

= Supported Plugins =
The following plugins are supported by the Factuursturen plugin:

* WooCommerce

== Installation ==

= To obtain API Key: =

* Log in to your factuursturen.nl account
* Go to your settings (left in the menu)
* Click on the option "Connections" (verbindingen)
* If you didn't create a key yet, click on the button "Create API key" (maak API sleutel) and fill the form
